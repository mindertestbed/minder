Features

Open Source

Minder is an open source online control flow management application (testbed) licenced under EUPL1.1

Stable

The Powerful Play-Scala-Java combination cooked with our teams experience provides a stable architecture that is immune to performance drawbacks.

Modular

The modular development framework enables arbitrarily complex inter-system communication to be handled easily

Powerful

The SCALA Based MTDL scripting capability with the help of additional maven dependency support provides an extremely powerful testing engine

Sustainable

An online scripting capability and open source modular Minder development framework helps create a sustainable architecture

User Friendly

Minder provides an easy to explore data model that is based on grouping of the inventories
