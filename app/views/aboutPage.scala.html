Minder The Online Programmable Flow Control Engine

Minder is a generic online programmable flow control engine that provides capability of interconnecting different systems in one node and enabling architects to perform complex communication operations on those interconnected systems.





Version Info

The current version of Minder is @{ utils.Util.getVersionInfo() }. Minder supports GITB service level compatibilty since V2.0-J15. The minder source code is hosted under the GitHub repository at https://github.com/mindertestbed.

Minder Team

Minder was developed as an open source project in eSENS to facilitate a sustainable and reusable conformance and interoperability testing. Further details available at https://joinup.ec.europa.eu/software/minder/description.
